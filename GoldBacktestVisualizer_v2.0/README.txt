MT4 Gold Backtest Visualizer v2.0
==================================
変更点:
- 移動平均線追加（EMA/SMA切替、3本まで、期間・色指定可能）

使い方:
1. index.html をダブルクリックしてブラウザで開く
2. HTMLファイルを選択
3. チャート上に全取引が表示されます

MT4でのHTMLエクスポート:
バックテスト: ストラテジーテスター > 結果タブ > 右クリック > レポート保存
口座履歴: MT4ターミナル下部 > 口座履歴タブ > 右クリック > レポート保存

動作: Google Chrome / Microsoft Edge 推奨
インターネット接続不要（完全オフライン動作）
