MT4 Gold Backtest Visualizer v3.0
==================================
変更点:
- 時間足切り替え追加（M1/M5/M15/M30/H1/H4/D1）
- 読み込んだデータのベース時間足を自動検出
- ベース時間足より細かい時間足ボタンは自動無効化

v2.0からの変更点:
- 移動平均線（EMA/SMA、3本、期間・色指定可能）

使い方:
1. index.html をダブルクリックしてブラウザで開く
2. OHLCデータCSVを読み込む（M1推奨）
3. 時間足ボタンで表示切替
4. バックテストHTMLを読み込んで取引を確認

動作: Google Chrome / Microsoft Edge 推奨
インターネット接続不要
