使い方
① OHLCデータを用意する
MT4メニュー → ツール → 履歴センター → GOLD/USD → 該当時間足を選択 → エクスポート

フォーマット例：
2024.01.15,10:00,2048.50,2055.20,2045.30,2052.80,12345

② 取引データを用意する
MT4 → ターミナル → 口座履歴タブ → 右クリック → 「全履歴を保存」

ただしMT4のHTML形式をCSVに変換する必要があります。下記ヘッダーで自作でもOK：

Ticket,OpenTime,Type,Lots,OpenPrice,CloseTime,ClosePrice,Profit

Profit がプラスなら ✓ 利確、マイナスなら ✗ 損切りとして自動判定します。
